**标题：** 持仓助手 — 一个聚合多券商账户的跨平台持仓管理工具

---

分享一个自己做的小工具，解决了我自己炒股多年的一个痛点：**同时用好几个券商 App，来回切换看持仓，算总盈亏特别麻烦。**

## 做了什么

「持仓助手」是一个跨平台的持仓聚合管理工具，把你散落在不同券商的持仓统一到一个界面里。

- **跨平台聚合** — 富途、老虎、雪球、支付宝基金、A 股券商、期货，全部汇总
- **实时行情** — A 股、港股、美股、基金、商品期货的实时/延迟报价
- **智能盈亏** — 自动汇率折算，统一以人民币展示总盈亏和收益率
- **做空支持** — 期货做空仓位的盈亏也能正确计算
- **交易记录** — 加仓/减仓记录，已实现盈亏追踪
- **饼图分析** — 按股票/按平台的持仓分布一目了然
- **历史曲线** — 每日自动快照，查看组合的历史走势
- **数据本地存储** — 你的持仓数据不会上传到任何服务器

## 技术栈

- Flutter（支持 Web / Android / macOS）
- Drift (SQLite) 本地数据库
- Riverpod 状态管理
- Supabase 登录认证
- Vercel 部署 Web 版 + API
