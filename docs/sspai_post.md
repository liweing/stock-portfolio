## 为什么做这个工具

作为一个在多个平台都有持仓的投资者，我长期面临一个问题：A 股在一个券商，港股在富途，美股在老虎，基金在支付宝，期货又是另一个账户。

每次想看看「我到底赚了多少，亏了多少」，就得打开四五个 App，手动加总，再换算汇率。特别是港股和美股，每次都要心算一下折成人民币是多少。

于是我决定自己做一个工具来解决这个问题。

## 持仓助手是什么

「持仓助手」是一款跨平台的持仓管理工具，核心功能就是一句话：**把你散落在各处的持仓，聚合到一个界面里，统一看盈亏。**

### 核心功能

**1. 跨平台聚合**

支持录入来自不同券商/平台的持仓：富途、老虎、雪球、A 股券商、支付宝基金、期货账户等。所有持仓统一展示，按平台分组。

**2. 实时行情 + 自动汇率**

接入行情数据接口，实时获取 A 股、港股、美股、基金净值、商品期货的报价。不同币种自动折算为人民币，总盈亏一目了然。

**3. 智能盈亏分析**

- 累计盈亏：从你录入的成本价自动计算
- 今日盈亏：基于昨收价计算当日变动
- 收益率：按成本计算百分比
- 做空支持：期货做空仓位的盈亏也能正确计算

**4. 可视化分析**

- 饼图展示按股票/按平台的持仓分布
- 每日自动快照，生成组合历史走势图
- 各平台独立统计卡片

**5. 交易记录**

支持记录加仓和减仓操作，卖出时自动计算已实现盈亏。

### 隐私设计

所有持仓数据存储在你的设备本地（SQLite），不会上传到任何服务器。没有广告，没有追踪，没有数据分析。你的投资数据只属于你自己。

## 技术实现

项目使用 Flutter 开发，一套代码同时支持 Web、Android 和 macOS。

- **前端**：Flutter + Riverpod 状态管理 + Material 3
- **数据库**：Drift（SQLite ORM），数据完全本地化
- **认证**：Supabase Auth（仅用于登录，不存储持仓数据）
- **部署**：Vercel（Web 版 + Cron 定时快照）
